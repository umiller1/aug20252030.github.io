Other Elements

<img src="..." alt="...">

Image: Adds an image to the page. Requires the src attribute and alt attribute. Void element, no closing tag required.

inline

<hr>

Horizontal Rule: Create horizontal line across container.

block

<br>

Line Break: Create a line break.

block

<ul>...</ul>

Unordered List: Semantic container for lists without chronological or specific order. May only have ol, ul, and/or li elements as a direct child.

block

<ol>...</ol>

Ordered List: Semantic container for lists with a chronological or specified order. May only have ol, ul, and/or li elements as a direct child.

block

<li>...</li>

List Item: Semantic child element of a list element. Can only be used in an ol or ul as the direct child.

block

CSS

CSS Selectors & Combinators

div {...}

HTML Element Selector: Selects any and all elements whose tag/element name matches the selector name.

.card {...}

Class Selector: Selects any and all elements whose class attribute value matches the selector name.

#logo {...}

ID Selector: Selects the one element whose ID attribute value matches the selector name.

div p {...}

Descendant Selector: Selects the descendant element. The element to the right of the space is what is targeted and is the descendant. The element to the left of the space is the ascendant. The example targets p elements that are descendants of div elements.

div > p {...}

Child Selector: Selects the direct child(ren) element. The element to the right of the greater than symbol is a child of the element to the left of the greater than symbol. The example targets p elements that are children of div elements.

h1,
h2,
a {...}

Selector List: Selects a group of related or unrelated elements to target them separately under the same rule and share properties. The example targets h1 elements, and/or h2 elements and/or anchor elements.

div#hero

Compound Selector: Selects elements only if they have the corresponding attached attribute(s). The example targets the div element only if it also has an ID attribute value of hero.
