Container Elements

<div>...</div>

Division: General Container

block

<span>...</span>

Content Span: Inline General Container.

inline

<section>...</section>

Section: Semantic Container. Requires heading child.

block

<header>...</header>

Header: Semantic Container. Must be at the top of its container/Msut be first child.

block

<nav>...</nav>

Navigation: Semantic Container. Requires at least one anchor element or button element.

block

Text Elements

<h1>...</h1>

Heading: Semantic heading. H1-H6.

block

<p>...</p>

Paragraph: Paragraph element. Allows short sentences and paragraphs.

block

<a href="#">...</a>

Anchor: Hyperlink element used to link to sections on a page, a different page, or a different website. Must use href attribute.

inline

<b>...</b>

Bold: Bold element used to thicken text weight.

inline

<i>...</i>

Italic: Italic element used to slant text for emphasis.

inline

<strong>...</strong>

Strong: Strong element used to thicken text weight. Semantic version of the bold element.

inline

<em>...</em>

Emphasis: Emphasis element used to slant text for emphasis. Semantic version of the italic element.

inline

<sup>...</sup>

Superscript: Superscripted element used to create exponents.

inline

<sub>...</sub>

Subscript: Subscripted element used to create footnotes.

inline

<blockquote>...</blockquote>

Blockquote: Blockquote element used for special quotes.

block

Other Elements

<img src="..." alt="...">

Image: Adds an image to the page. Requires the src attribute and alt attribute. Void element, no closing tag required.

inline

<hr>

Horizontal Rule: Create horizontal line across container.

block

<br>

Line Break: Create a line break.

block

<ul>...</ul>

Unordered List: Semantic container for lists without chronological or specific order. May only have ol, ul, and/or li elements as a direct child.

block

<ol>...</ol>

Ordered List: Semantic container for lists with a chronological or specified order. May only have ol, ul, and/or li elements as a direct child.

block

<li>...</li>

List Item: Semantic child element of a list element. Can only be used in an ol or ul as the direct child.

block
